# Car Rental
Java implementation of CarRental service.

## Technologies
* Java 17
* Spring Boot
* MySQL
* Liquibase
* JUnit 5
* Mockito

## Getting started
To run this project:
1. Clone repository `git clone https://github.com/Hill-Coding/car-rental`
2. Compile and run project with Java
